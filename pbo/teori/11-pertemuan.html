Overload


public class hitung {
  public int tambah(int a, int b) {}
  public float tambah(float a, float b) {}
  public int tambah(int a, int b, int c) {}
  public float tambah(float a, float b, float c) {}
  public int tambah(int data[]) {}
}

Method Overload

  • Nama method harus sama.
  • Daftar parameter (argumen) harus berbeda (jumlah atau tipe data)
  • Return type boleh sama, ataupun berbeda.

Parameter pada Overloading

  • Perbedaan daftar parameter bukan hanya terjadi pada perbedaan parameter, tetapi juga urutan dari tipe data parameter.
  • Daftar parameter tidak terkait dengan penamaan variabel yang ada dalam parameter

Trivia


public void fungsi(int x, String s) {}


public void fungsi(String s, int x) {}

Trivia


public void fungsi(int x) {}


public void fungsi(int y) {}

Constructor Overloading

  • Seperti halnya pada method, dapat dilakukan overloading pada constructor
  • Dapat digunakan keyword this untuk memanggil constructor yang lain

public class Mahasiswa {
  private String nim;
  private String nama;
  private String gender;

  public Mahasiswa(String nim, String nama) {
    this(nim, nama, null);
  }
  public Mahasiswa(String nim, String nama, String gender) {
    this.nim = nim;
    this.nama = nama;
    this.gender = gender;
  }
}
